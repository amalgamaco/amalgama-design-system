Ring de error

VariableValorUso
--color-error-ringcolor-mix(in srgb, var(--color-error) 12%, transparent)Ring de error para inputs (.is-error). Derivado de --color-error via color-mix, por lo que se recalibra solo en dark. Nunca hardcodear el rgba.

Breakpoints

Referencia canónica del DS. Limitación CSS: las media queries no pueden consumir var() — usá los valores literales en @media y mantenelos sincronizados con estos tokens.

VariableValorUso
--breakpoint-md768pxDebajo: layout de columna única; grids colapsan
--breakpoint-lg1024pxDebajo: contenido compacto; sidebar puede colapsar
