Chip

Elementos compactos que representan una accion, filtro, entrada o sugerencia. Cuatro variantes funcionales: assist, filter, input y suggestion. Alto 32px, radio full, tipografia Inter 500 14px.

CSS: css/components/chip.css · React: components/ui/chip.tsx (Chip, InputChip, ChipSet)

Cuándo usar

Cuándo usarFiltros, selección múltiple, sugerencias — siempre interactivo. Filter chips van debajo de la search bar para refinar resultados.
Cuándo noEstado de solo lectura (usar Badge); acción principal (usar Button).
Reemplaza aToggles de filtro ad-hoc, tags clickeables legacy.

Las cuatro variantes

ana@amalgama.co 
<!-- ASSIST: accion contextual, con icono inicial -->
<button class="chip">
  <span class="chip-icon"><svg>…</svg></span>
  Agendar entrevista
</button>

<!-- FILTER: refina contenido, alterna aria-pressed + .chip-selected -->
<button class="chip" aria-pressed="false">Remoto</button>
<button class="chip chip-selected" aria-pressed="true">Full-time</button>

<!-- INPUT: informacion ingresada, con boton de eliminacion -->
<span class="chip">
  ana@amalgama.co
  <button class="chip-remove" aria-label="Quitar ana@amalgama.co"><svg>…</svg></button>
</span>

<!-- SUGGESTION: sugerencia dinamica -->
<button class="chip">Buscar diseñadores</button>

Elevado y deshabilitado

<button class="chip chip-elevated">Elevado</button>
<button class="chip" disabled>Deshabilitado</button>

Conjunto de chips (filtros)

<div class="chip-set" role="group" aria-label="Filtrar vacantes">
  <button class="chip" aria-pressed="false">Diseño</button>
  <button class="chip chip-selected" aria-pressed="true">Ingenieria</button>
  <button class="chip" aria-pressed="false">Producto</button>
</div>

<script>
  document.querySelectorAll('.chip[aria-pressed]').forEach(function (chip) {
    chip.addEventListener('click', function () {
      var on = chip.getAttribute('aria-pressed') === 'true';
      chip.setAttribute('aria-pressed', String(!on));
      chip.classList.toggle('chip-selected', !on);
    });
  });
</script>

React

import { Chip, InputChip, ChipSet } from "@/components/ui/chip"

<ChipSet label="Filtrar vacantes">
  <Chip selected={filtro === 'remoto'} onClick={() => toggle('remoto')}>Remoto</Chip>
  <Chip icon={<CalendarIcon />}>Agendar entrevista</Chip>
  <Chip variant="elevated">Elevado</Chip>
  <InputChip onRemove={() => quitar(email)} removeLabel={`Quitar ${email}`}>
    {email}
  </InputChip>
</ChipSet>
