Search

Busqueda de contenido. Dos variantes: search bar (campo persistente de 56px con forma de pildora) y search view (vista expandida con resultados, en modo docked o full-screen).

CSS: css/components/search.css · React: components/ui/search.tsx (SearchBar, SearchView y sub-componentes)

Para busqueda compacta embebida en una toolbar usa .search-field (Toolbar). Para refinar resultados, combina con filter chips debajo de la search bar.

Cuándo usar

Cuándo usarsearch-bar standalone (56px, pill) para vistas de búsqueda dedicadas; search-view para resultados. Combinar con filter chips debajo.
Cuándo noBúsqueda compacta dentro de una toolbar (usar .search-field de toolbar.css).
Reemplaza aInputs de búsqueda legacy.
