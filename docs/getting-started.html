Guia de uso

Todo lo que necesitas para empezar a usar Amalgama DS en tu proyecto. Sin npm, sin build tools — solo HTML y CSS.

1. Instalacion

Agrega las fuentes de Google Fonts y los 4 archivos CSS al <head> de tu HTML. No se necesita ningun otro paso.

<!-- Google Fonts -->
<link rel="preconnect" href="https://fonts.googleapis.com">
<link href="https://fonts.googleapis.com/css2?family=DM+Mono:wght@400;500&family=Epilogue:wght@400;500;600;700&family=Inter:wght@300;400;500;600;700&display=swap" rel="stylesheet">

<!-- Amalgama Design System -->
<link rel="stylesheet" href="css/variables.css">
<link rel="stylesheet" href="css/base.css">
<link rel="stylesheet" href="css/layout.css">
<link rel="stylesheet" href="css/components.css">
ArchivoContenidoLineas
variables.cssDesign tokens: colores, radios, sombras, transiciones82
base.cssReset, tipografia, animaciones, clases utilitarias81
layout.cssApp shell: sidebar, topbar, contenido189
components.cssBarrel file que importa todos los componentes—

Opcion B: Solo los componentes que necesitas

En vez de cargar todos los componentes, copia solo los archivos CSS que necesitas desde css/components/:

<!-- Core (requerido) -->
<link rel="stylesheet" href="css/variables.css">
<link rel="stylesheet" href="css/base.css">

<!-- Solo los componentes que usas -->
<link rel="stylesheet" href="css/components/button.css">
<link rel="stylesheet" href="css/components/form.css">
<link rel="stylesheet" href="css/components/table.css">
<link rel="stylesheet" href="css/components/badge.css">

3. Usar un componente

Cada componente es un archivo CSS independiente en css/components/. Copia el archivo CSS a tu proyecto, agrega el <link>, y pega el HTML. Cada archivo incluye un comment header con ejemplo de uso y dependencias.

1.
Buscar
Navega por los componentes en la barra lateral
2.
Copiar
Click en "Copiar" en el bloque de codigo
3.
Pegar
Pega el HTML en tu pagina — listo

4. Personalizar tokens

Cambia cualquier variable en variables.css y toda la interfaz se actualiza automaticamente. Estos son los tokens principales:

TokenDefaultControla
--accent#4f80ffBotones, links, estados activos, badges accent
--primary-500#253e82Color navy del brand, hover de botones
--radius12px (lg)Border-radius de cards, modals, tables
--radius-md8pxBorder-radius de inputs, botones, kanban cards
--bgvar(--neutral-50)Fondo de pagina
--card-bg#ffffffFondo de tarjetas, sidebar, topbar
--bordervar(--neutral-100)Todos los bordes del sistema
--shadow-md0 4px 16px rgba(28,36,56,.08)Hover en cards, dropdowns
--duration-fast120msTodas las transiciones rapidas (hover, focus)

Ejemplo: cambiar el color accent

/* variables.css — cambia solo esta linea */
--accent: #10b981;    /* de azul a verde */

/* Resultado: TODOS los botones, links, tabs activos,
   badges accent y focus rings cambian a verde */

ArchivoQue incluye
