# CLAUDE.md

This file provides guidance to Claude Code when working on the Amalgama Design System.

## What this repo is

This repository contains the Amalgama Design System documentation site.

The goal is to create a clear, scalable, and well-documented internal Design System for Amalgama products.

This is not only a component showcase. It should work as a real documentation tool for designers and developers, covering foundations, tokens, components, specs, behaviors, usage guidelines, accessibility, illustrations, UI animations, and content/writing principles.

## Core direction

Use mature design systems as structural references, especially:

- Material Design 3
- Uber Base Design System

Use them only as references for documentation architecture, navigation, token structure, component specs, behaviors, and usage guidelines.

Do not copy their visual style.

The Design System must preserve Amalgama’s identity: colors, typography, UI components, brand feel, tone, and visual language.
