Filosofia

Construido para velocidad de equipo, no para postales.

Un sistema que prioriza claridad sobre complejidad. Sin frameworks, sin runtime, sin sorpresas — solo decisiones de diseno bien documentadas.

Tokens primero

Colores, sombras, radios y tipografia expuestos como CSS variables. Una sola fuente de verdad, lista para temas y modos.

Copia y pega

Cada componente trae HTML, CSS y, cuando aplica, su contraparte en React. Sin instalaciones, sin configuracion.

Docs vivas

Los ejemplos en la documentacion son los componentes reales. Si funcionan ahi, funcionan en produccion.

Lo que esta adentro

Piezas que ya estan trabajando en produccion.

Cada componente nacio de una pantalla real — vacancies, kanbans, dashboards de candidatos. Lo que ves es lo que esta corriendo.

Accesibilidad por defecto. Contrastes WCAG AA, focus rings visibles, semantica HTML correcta.
Responsive sin esfuerzo. Cada componente se adapta de mobile a desktop usando solo CSS.
Espanol primero. Labels, mensajes y ejemplos en ES — sin tener que traducir despues.
